# Ionic & NodeJS Assignment

An Ionic application that redirect the user to the selected/default url.

## Installation

```sh
npm ci
```

## Usage

```sh
# Run Ionic client
npm run dev:client

# Run Express server
npm run dev:server

# Run both
npm run dev

# Run Ionic client using production api
npm run prod
```

## Built with

- Express.js v4
- express-validator
- Angular v5.2.11
- Ionic v3.9.2
